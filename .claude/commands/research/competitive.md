# Competitive Analysis: $ARGUMENTS

Identify and evaluate alternatives to **$ARGUMENTS**. Build a comparison matrix and recommend a strategy.

## 1. Identify Competitors/Alternatives
Find at least 3-5 competitors or alternative approaches. For each, gather:
- Name and brief description
- Target audience
- Maturity and market presence

## 2. Individual Assessment
For each competitor, evaluate:

| Aspect | Details |
|--------|---------|
| **Strengths** | What they do well |
| **Weaknesses** | Where they fall short |
| **Pricing/Cost** | Free, freemium, paid, enterprise |
| **Key Differentiator** | Their unique selling point |
| **Community/Support** | Docs quality, community size, responsiveness |

## 3. Comparison Matrix

| Feature | Our Approach | Competitor A | Competitor B | Competitor C |
|---------|-------------|-------------|-------------|-------------|
| (feature 1) | Y/N/Partial | Y/N/Partial | Y/N/Partial | Y/N/Partial |
| (feature 2) | ... | ... | ... | ... |
| (feature 3) | ... | ... | ... | ... |

## 4. Gap Analysis
- **They have, we don't**: Features or capabilities we are missing
- **We have, they don't**: Our unique advantages
- **Common ground**: Table-stakes features everyone shares

## 5. Strategic Recommendation
- Which competitive position should we target? (cost leader, differentiator, niche)
- Top 3 features to prioritize based on competitive gaps
- What would make us the clear winner in this space?
